## Interactive Elements

### Click Actions
- **Keywords:** Click to copy to clipboard (shows "âœ“ Copied!" toast)
- **Filter Buttons:** Toggle between All/Covered/Not Covered requirements
- **Expandable Sections:** (Future) Click to expand/collapse long content

### Hover Effects
- **Cards:** Subtle lift and glow on hover
- **Buttons:** Color change and background highlight
- **Keywords:** Border color changes to primary color
- **Copy Button:** Tooltip appears

### Visual Indicators
- **Loading:** Animated spinner with message
- **Empty State:** Icon + helpful message + suggested action
- **Status Badges:** Color-coded (green=covered, yellow=not covered, red=must-have, blue=nice-to-have)
- **Progress Bars:** Animated fill based on percentage
- **Highlights:** Keywords in yellow, changes in green/red backgrounds
